# Editor Plugins
---------------------

You can boost your developer experience with Hegel by editor integrations.
Hegel try to provide an editor support for: Types hints and Errors display.

Plugins:
- [Visual Studio Code](#visual-studio-code)

## Visual Studio Code

You can install by [Visual Studio Code Market](/) or manualy.

```bash
$ git clone https://github.com/JSMonk/hegel
$ cd hegel/packages/language-server
$ npm run complie
$ code --install-extension ./hegel-plugin.vsix
```
